Tasbeeh Counter App

A beautiful and interactive Tasbeeh Counter App built using HTML, CSS, and JavaScript.
This app helps users perform digital tasbeeh counting with sound effects and changing images.

Features
Tasbeeh count increment on button click
Reset counter functionality
Click sound effect on every count
Automatic image change after certain counts
Responsive and attractive UI
Beginner-friendly JavaScript project
Technologies Used
HTML5
CSS3
JavaScript (DOM Manipulation)
